## About

我主要关注移动端产品体验与工程实现之间的平衡，长期投入在 iOS 原生开发、Flutter 跨平台应用，以及媒体工具和自动化类产品的界面与交互设计。

相比单纯堆叠功能，我更在意页面结构是否清晰、操作路径是否自然，以及项目在持续迭代中的可维护性。

## Focus

- iOS application development with `Swift` and `SwiftUI`
- Cross-platform product delivery with `Flutter`
- Media workflow, automation tools, and utility apps
- UI clarity, interaction rhythm, and engineering maintainability

<p>

这个项目更像一次完整的移动端产品化实践：不是简单移植桌面端信息，而是围绕通知、任务、媒体状态和日常操作频率，重新组织页面结构与交互方式。
